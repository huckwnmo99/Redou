# Manual Smoke Checklist

초기 앱 셸 확인용 수동 체크리스트입니다.

## UI Shell

- 앱이 21번 스타일 기반 레이아웃으로 렌더링된다.
- 좌측 sidebar에 workspace nav와 category tree가 모두 보인다.
- 상위 카테고리와 하위 카테고리가 구분되어 보인다.

## Category Flow

- `+ Top`으로 새 상위 카테고리를 추가할 수 있다.
- `+ Sub`으로 현재 상위 카테고리 아래 새 하위 카테고리를 추가할 수 있다.
- 하위 카테고리를 바꾸면 Library와 Category View의 내용이 같이 바뀐다.

## Paper Flow

- Library 카드에서 `Open Detail`을 누르면 논문 상세로 이동한다.
- `Open Reader`를 누르면 리더 화면으로 이동한다.
- 논문 상세에서 현재 카테고리가 노출된다.

## Search And Settings

- 검색어를 바꾸면 Search 화면의 문구가 갱신된다.
- Settings 화면에서 account, backup, trash, workspace 상태가 보인다.
