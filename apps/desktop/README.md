# Desktop App

Redou의 데스크톱 앱 시작점입니다.

## Planned Stack

- `Electron`
- `React`
- `TypeScript`
- `Vite`

## Initial Commands

의존성 설치 후 아래 순서로 실행합니다.

```bash
npm install
npm run dev
```

Electron 셸을 별도로 확인할 때:

```bash
npm run start:electron
```

개발 단계에서는 Vite dev server 기본 포트 `5173`을 기준으로 Electron이 renderer를 불러오도록 설정했습니다.
