# Redou Agent Slide Assets

These SVG assets are for the first three slides of the beginner-friendly Redou agent presentation.

## Files

- `slide-01-hero-illustration.svg`
  - Use on slide 1.
  - Best placement: right-side card or as a large supporting graphic on the lower-right half.
  - Purpose: show the big picture flow `PDF -> structure -> agent -> Q&A/table`.

- `slide-02-problem-illustration.svg`
  - Use on slide 2.
  - Best placement: center or lower half.
  - Purpose: visually explain the three pain points: too many PDFs, forgotten content, manual comparison.

- `slide-03-flow-illustration.svg`
  - Use on slide 3.
  - Best placement: main central visual.
  - Purpose: show the simple product flow `upload -> structure -> save -> ask/compare -> use`.

- `slide-04-ai-overview.svg`
  - Use on slide 4.
  - Purpose: show that Redou uses multiple AI parts together, not just one model.

- `slide-05-qa-vs-table.svg`
  - Use on slide 5.
  - Purpose: visually compare the short Q&A path with the longer table-generation path.

- `slide-06-why-multi-ai.svg`
  - Use on slide 6.
  - Purpose: explain why OCR, embedding, LLM, and verification are all needed.

- `slide-07-agent-loop.svg`
  - Use on slide 7.
  - Purpose: explain the beginner-friendly agent loop: observe, plan, act, check.

- `slide-08-table-agent-flow.svg`
  - Use on slide 8.
  - Purpose: show the detailed table-mode agent pipeline.

- `slide-09-qa-flow.svg`
  - Use on slide 9.
  - Purpose: show the simpler Q&A pipeline.

- `slide-10-agentic-vs-chatbot.svg`
  - Use on slide 10.
  - Purpose: compare a normal chatbot with agentic AI in a classroom-friendly way.

## Styling Notes

- Matches the existing Figma deck direction:
  - light background
  - blue / green / orange accents
  - rounded SaaS-style cards
  - minimal educational illustration style
- Fonts referenced in the SVGs:
  - `Noto Sans KR`
  - `Pretendard`
  - fallback `sans-serif`

## Suggested Import Workflow

1. Open the Figma deck: `Redou Agent Presentation`
2. Import each SVG into its matching slide
3. Keep the illustrations slightly smaller than the text blocks so the deck stays easy to read
4. If needed, lower opacity of decorative assets to 90-95% for a cleaner classroom presentation look
