Layout language first

Six designs that actually change the interface grammar.

This set stops trying to be one family. Each concept uses a different UI language: collector, sharp split, minimal reader, window system, board, and table.

21 Reference Collector
Closest to your screenshot reference. Flat, clean, card-based.
 22 Sharp Split
Hard edges, clear divisions, very sharp and product-like.
 23 Minimal Reader
Almost editorial. Most whitespace. Least dashboard feeling.
 24 Windowed Studio
Detachable-window idea treated as the main design concept.
 25 Canvas Board
Column board for notes, figures, and folders.
 26 Index Table
Table/list-first tool for heavy catalog management.
